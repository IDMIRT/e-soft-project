Задача:

Создайте API-сервис со следующей функциональностью:

Загрузка файлов (CSV, Excel) через API.
Анализ данных с помощью Pandas (вычисление среднего значения, медианы, корреляции).
Хранение загруженных файлов и результатов анализа в PostgreSQL.
Предоставление эндпоинтов для получения статистики по данным.
Очистка данных (удаление дубликатов, заполнение пропущенных значений).
Задание продвинутого уровня, вы можете выполнить его по своему желанию: добавьте функционал, который реализует построение графиков и отчетов в вашем API.

Технические требования:

Python 3.7+
Flask
Pandas
SQLAlchemy
PostgreSQL
REST API
CRUD операции
GitHub для хранения кода

Разработка API-сервиса включает в себя следующие ключевые аспекты:

Проектирование архитектуры API с использованием Flask.
Реализация эндпоинтов для загрузки файлов, анализа данных и получения статистики.
Интеграция с PostgreSQL для хранения файлов и результатов анализа.
Использование Pandas для обработки и анализа данных.
Реализация функционала очистки данных.
Обработка ошибок и возврат соответствующих HTTP-статусов.

При выполнении задания вы можете использовать любые дополнительные библиотеки Python без ограничений. Они должны быть добавлены в проект через pip, если библиотека поддерживает такой способ установки.

Сроки реализации задачи — 1 неделя (9 часов чистого времени).

Самопроверка

Для контроля выполнения задания проверьте следующие пункты:

API позволяет загружать файлы (CSV, Excel) и сохраняет их в PostgreSQL.
Реализован анализ данных с помощью Pandas (среднее, медиана, корреляция).
Есть функционал очистки данных (удаление дубликатов, заполнение пропущенных значений).
API корректно обрабатывает ошибки загрузки файлов (400 Bad Request, 415 Unsupported Media Type).
Реализованы следующие эндпоинты:
POST /upload для загрузки файла
GET /data/stats для получения аналитики по загруженным данным
GET /data/clean для очистки данных
(Продвинутый уровень) GET /data/plot для получения графика

1 Проработка /uploads не загружает файл и не дает ошибки
