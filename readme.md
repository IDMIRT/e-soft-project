## Разработка для сервиса E-SOFT

API - сервис для загрузки и хранения данных в формате csv/xls/xlsx

## Установка и запуск

1. Клонируйте репозиторий.

2. Подготовка БД

    При установленном docker: 

    запустите docker volume create pg_data - для создания тома хранения данных
    
    запустите docker-compose.yaml в папке приложения

    Без установленного docker:

    Установите Postgres проверьте что он работает по порту 5432 

3. Создайте виртуальное окружение, активируйте его, после входа в виртуальное 
окружение установите библиотеки из requirements.txt

4. Для тестирования запустите файл run.py находясь в папке приложения или используя вебсервер
запустите приложение, например gunicorn

после запуска можно тестировать

загрузка файла curl -X POST -F "file=@myfile.csv" http://localhost:5000/api/upload

очистка файла - http://localhost:5000/api/data/clean/1

получение статистики curl http://localhost:5000/api/data/stats/1

генерация графика curl http://localhost:5000/api/data/plot/1 -o graph.png
